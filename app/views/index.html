View a Repository

Click the button to view a user's repository by specifying their username and repository name.

Remember that only text files can be viewed. If a non-text file is in the repository, the view will not update properly.

Connect to a Repository

Click the button to connect to a user's repository by specifying their username and repository name, and pointing to a directory of your choice. The repository will then be stored in the chosen directory and will be a perfect reflection of the original.

Remember that any edits you make to the files in the repository will be overwritten if the owner edits the file. This differs from viewing since all file types will update properly (not only text files). You can connect to multiple repositories.

Broadcast a Repository

Click the button to broadcast a directory of your choice. You must be logged in to access this feature. Once a directory is broadcasted, it will be able to be viewed by anyone who knows your username. You can broadcast multiple directories.

As of right now, a maximum of 500 files is allowed per directory, and each file must be no larger than 5MB (this may change in the future). A .gitignore file can be created to prevent certain files and directories from being broadcasted. This file should follow the exact same format as a normal .gitignore file for GitHub.
